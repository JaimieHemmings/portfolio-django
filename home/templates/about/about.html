Tech Stack

My technical skills include:

Front End

HTML

I've used HTML to create clean, semantic markup that ensures websites are well-organized, accessible, and responsive.

CSS

I have strong experience with CSS, using it to style and layout web pages for a polished, visually appealing user experience. I also have extensive experience with a variety of CSS frameworks and Libraries.

JavaScript

I have utilized JavaScript to build responsive interfaces, handle user interactions, and manage real-time data updates. I'm well-versed in modern JavaScript features such as ES6+, asynchronous programming (promises, async/await), and DOM manipulation.

React

I've used React to develop component-based architectures that improve performance and scalability. On the back end, I've used JavaScript with Node.js to build APIs, handle server-side logic, and manage databases, providing a seamless connection between the front-end and back-end of your applications.

Back End

Python

I have significant experience with Python, particularly for back-end development. I've used Python to build robust server-side logic, develop RESTful APIs, and manage data flow between the front end and databases.

PostGreSQL

Using PostgreSQL, I've utilized its relational database structure to design normalized schemas, manage complex queries, and ensure data integrity through constraints and transactions. I've worked with SQL to perform CRUD operations, optimize queries, and implement indexing to improve performance for data-intensive applications.

MongoDB

My experience with MongoDB, a NoSQL database, involves handling unstructured or semi-structured data. I've leveraged its document-based structure to work with flexible schemas, allowing rapid iteration and scalability, particularly for applications with changing data models.

Testing

Cypress

I use Cypress for end-to-end testing to ensure that web applications function as intended from the user's perspective. Cypress allows me to write automated tests that simulate real user interactions, such as clicking buttons, filling out forms, or navigating through the site.

Jest

As a full stack developer, I use Jest for unit testing and ensuring the reliability of my JavaScript code. Jest allows me to write and run tests for individual components or functions in isolation, helping me verify that each piece of my application behaves correctly. I utilize Jest's features like test mocks, spies, and assertions to simulate different scenarios and validate how the code should respond under various conditions.
